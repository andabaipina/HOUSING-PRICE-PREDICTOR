# HOUSING-PRICE-PREDICTOR


Predicting house sale prices using the Ames Housing dataset.

## What I did
- Cleaned and explored 2,900+ rows of real housing data
- Engineered 4 new features including house age and total square footage
- Trained and compared Linear Regression vs Random Forest models
- Visualised predictions using interactive Plotly charts

## Tools used
Python, pandas, scikit-learn, Plotly, matplotlib
